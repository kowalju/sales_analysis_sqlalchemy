# sales_analysis_sqlalchemy
A **Python** project for managing and analyzing sales data using **Pandas** and **SQLAlchemy ORM**.<br>
The script:
- loads data from CSV file,
- calculates the total sales value for each product,
- saves the data to a SQLite database using SQLAlchemy ORM,
- indentifies the top 5 best-selling products,
- writes the results to a text reprt.
